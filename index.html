About

Programming is a vital skill to have as a geoscientist. As we delve into sophisticated research questions, we need specialized tools to analyze and manage huge amounts of data to help solve problems and explore ideas. In addition, the culture of academia is one centered around being self-taught, and although that may be standard for some academics, it does not necessarily apply to everyone. It is especially difficult to try to learn a new skill in a language not native to one’s own.

The GeoLatinas Coding Group initiative was born out of the idea to provide this widely-needed skill tool set to the members of the Latinas in Geoscience organization. In this way, we provide a welcoming environment to learn to code for all GeoLatinas, regardless of one’s level of comfort with either English or Spanish. To maintain consistency, we provide two coding sessions a week (in Spanish) where we go over programming in Python.

If you wish to know more about Coding Group initiative and GeoLatinas community, we invite you go to GeoLatinas blog and GeoLatinas website.

Getting Started

Setup your System

In order to start doing our courses you need to install a Python distribution. We recommend using Anaconda. You can find detailed instructions on how to install it on any operating system in the Software Carpentry setup instructions .

New to Python?

If you don’t have much experience with Python, don’t worry!. A great place to start learning Python for scientific computation and much more is Software Carpentry. We recommend starting with these two lessons (also available in Spanish):

  • Programming with Python
  • Plotting and Programming in Python
